    2025年12月アップデート内容

    • 新機能
      • お知らせ機能の追加(Web版/スマホ版)
        • 管理者が全ユーザに対してお知らせを配信できるようになりました
          • 部署や事業所ごとに配信対象を絞ることも可能です
        • お知らせの作成・編集・削除は管理者画面から行えます
        • お知らせはアプリ起動時にポップアップで表示されます
      • スーパーグラッチェの配布コインを送信ユーザにも付与する機能の追加(Web版/スマホ版)
        • スーパーグラッチェを送信したユーザにもコインを付与できるようになりました
        • 管理者画面からスーパーグラッチェ送信者へのコイン付与額を入力することでご利用いただけます
    • 変更点
      • オンラインマニュアルがもっと手軽にご覧いただけるようになりました！(Web版/スマホ版)
        • メニュー内にオンラインマニュアルへのリンクが追加されました
        • これまで以上に簡単にマニュアルをご覧いただけます
      • グラッチェするユーザ一覧では一覧全体をタップして選択できるようになりました(スマホ版)
        • これまでのように👍をタップする方法に加えて、ユーザ名やサムネイル部分をタップしても選択できるようになりました

    2025年9月アップデート内容

    • 新機能

      • グラッチェの大きさ設定機能(Web版/スマホ版)

        • 管理者画面からグラッチェの大きさ設定を有効にすることでご利用いただけます
        • グラッチェをされたユーザに付与するGPを変更することができます
        • ※グラッチェしたユーザは従来通り1GP固定となります

      • メールアドレスの表示設定機能(Web版/スマホ版)

        • 管理者画面からメールアドレスを非表示にすることでご利用いただけます
        • 非表示に設定すると、一般画面で全ユーザのメールアドレスが表示されなくなります

      • ユーザー登録リンクのQRコードダウンロード機能(Web版：画面項目(#5))

        • ユーザ登録リンクをQRコードでダウンロードできるようになりました

    2025年5月アップデート内容

    • 新機能

      • グラッチェ取消(Web版/スマホ版)

        • 管理者画面からグラッチェ取消を許可することでご利用いただけます
        • メッセージ送信後、24時間以内であれば取消ができるようになります
        • 取り消されると履歴に表示されなくなります
        • 付与されたグラッチェポイントや、スーパーグラッチェで送られたコインは元に戻ります

      • ユーザ情報変更(Web版/スマホ版)

        • 管理者画面からユーザ情報変更機能を有効にすることでご利用いただけます

        • ユーザ自身でユーザ情報を変更できるようになりました

          ※現在変更できるのは「所属情報」のみです。

      • 自動集計実行時のメール送信(Web版)

        • 現在設定されている自動集計は、デフォルトで「メール送信」が有効になっています。メールを送りたくない場合は、編集画面から完了メールを無効にしてください
        • 自動集計が実行された際に、通知メールを送信するオプションを追加しました
        • 失敗した場合には、原因もあわせてメールに記載されます

    • 変更点

      • デフォルトメッセージ画面の初回タップ時に設定されているメッセージをクリアするようになりました(スマホ版)
        • 設定されているデフォルトメッセージを確認・編集しやすくなりました

      • コイン履歴に送金時のメッセージが表示されるようになりました(スマホ版)

        • 対象：スーパーグラッチェ／コイン送金／管理者画面からのコイン送金

      • ボタンが押せない場合は灰色で表示されるようになりました(スマホ版)

        • 押せない状態がひと目で分かるようになり、操作ミスを防ぎやすくなりました

          • 押せない状態

          • 押すことができる状態

      • 画面上部に「Grazie Coin」のロゴが表示されるようになりました(スマホ版)

        • 現在の画面が「Grazie Coin」の画面であることがより分かりやすくなりました

    2025年3月アップデート内容

    • 新機能
      • パスワード再発行機能の追加(スマホ版)
        • ログイン画面からパスワード再発行のメールを送信できるようになります
    • 変更点
      • ランキング画面の操作性向上(スマホ版)
        • 事業所の切り替え方法をタブからプルダウンに変更しました
      • パフォーマンス向上(スマホ版)
        • デザインを一部改善し、よりスムーズに操作できるようになりました

    2024年11月アップデート内容

    • 変更点
      • マスターアカウントの追加・変更・削除機能の実装(Web版)
        • 従来のマスターアカウントは一人のみでしたが、今回のリリースで複数人のマスターアカウントを作成できます
        • マスターアカウントの追加・変更・削除はマスターアカウントのみ操作できます
        • CSVインポートでマスターアカウントを追加することはできません
        • マスターアカウントは最低一人以上必要です
      • グラッチェ履歴からグラッチェできるようになります(Web版)
        • アプリ版と同様に、ポイント履歴画面から直接グラッチェできるようになります
      • Web版のサムネイル変更機能の刷新(Web版)
        • レイアウトを変更し、操作性が向上しました
        • アプリ版と同様に、画像の切り取り操作を行えるようになります
      • Web版の通知設定画面のレイアウト変更(Web版)
        • 視認性を向上しました
      • CSVインポート時の更新処理の変更(Web版)
        • 従来は更新時にメールを送信していましたが、今回のリリースからはメールを送信しないようになります ※従来通り追加時は送られます
      • 軽微な不具合を修正

    2024年10月アップデート内容

    • 新機能

      • コイン送金、回収する際に1~100文字でメッセージを入力できるようになりました(Web/アプリ)
        • メッセージを追加することで、コイン送金をする経緯を記録することができます
        • たまにはお世話になっている方にグラッチェ以外の方法で感謝をしてみませんか？

    • 変更点

      • 集計結果を表彰する際にコインを送金する必要がなくなりました(Web)
        • コイン送金することなく、一般ユーザに表彰結果を公開することができます
        • 手動/自動集計どちらともコイン送金なしで表彰することができます
      • ユーザCSVインポートする際に空行が含まれていてもインポートできるようになりました(Web)
        • 空行部分をアプリ側が自動的に排除することで、より快適にユーザインポートができるようになりました
        • ※従来では空行が含まれている場合は空行部分を手動で削除し、再インポートをする必要がありました
      • 利用許諾を同意する画面のレイアウトが変更されました(Web/アプリ)
        • さらに見やすくするため外部リンクに変更しました

    2024年9月アップデート内容

    • 新機能
      • 商品に在庫数を設定できるようになりました！(Web版/スマホ版)
        • 在庫数が0になると「売り切れ」になり、ユーザは商品の交換ができなくなります
      • 商品の交換申請を取り消しできるようになりました！(Web版/スマホ版)
        • 取り消しを行うと、交換時に消費したコインが返金されます
        • ※2024年9月以降に交換申請した商品のみが対象です
      • コイン送金機能の制限ができるようになりました！(Web版/スマホ版)
        • 制限されたユーザは一般画面のコイン送金ができなくなります(管理者画面からのコイン送金は除く)
      • CSVアップロードが更新にも対応するようになりました！(Web)
        • 今までは登録のみでしたが、ついに更新にも対応しました
        • また、現在登録しているユーザの一覧をCSVでダウンロードできるため、更新も簡単です！
      • 履歴からグラッチェできるようになりました！(スマホ版)
        • 履歴に表示されているユーザ名をタップするだけで、グラッチェ入力画面に遷移します

    2024年7月アップデート内容

    • 変更点

      • スマホアプリ版のみ

        • 設定画面からアプリの言語設定ができるようになりました！

          • 現在は日本語と英語のみとなります
          • デフォルトでは端末の言語設定に依存します

        • ホーム画面と設定画面のレイアウトを変更しました！

          • ユーザ情報を設定画面に移動することでホーム画面に表示されている直近5件の履歴が見やすくなります

        • ユーザのサムネイルのアップロード時に確認画面を表示するようになりました

          • 今までは画像を選択すると即座にアップロードしていたため、確認画面を追加することで間違った画像が設定されることは無くなりました

      • スマホアプリ/Web版

        • 事業所/部署名の最大文字数を20→50文字まで拡張しました！
          • 画面表示部分に入りきらない場合は省略されます
          • Web版のみマウスカーソルすると全文字が表示されます
          • 登録方法については組織をご確認ください

    • 不具合の修正

      • スマホアプリ/Web版
        • グラッチェメッセージに絵文字を使用すると文字化けする不具合を修正
          • 端末によって表示されない絵文字がある場合もありますがご了承ください。

    2022年9月アップデート内容

    • 管理者コンソールの画面構成をメニュー形式にしました
    • 管理者コンソールに以下の機能を追加しました
      • スーパーグラッチェ
        • グラッチェをするとコインがもらえる機能です
      • グラッチェの回数制限
      • コインの一括配布・一括回収
      • 商品交換状況管理
      • 自動集計設定
      • 予算管理
        • コインの支出内容を確認できる機能です
      • 集計・表彰の自動化
      • 利用状況
        • ログインユーザ数、グラッチェ数をグラフ表示できます
      • ユーザ登録フォーム
