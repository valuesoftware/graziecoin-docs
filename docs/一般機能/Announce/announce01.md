# お知らせ掲示板画面

## ❓お知らせ掲示板画面とは

「お知らせ掲示板画面」は、一般ユーザが管理者からのお知らせを一覧で確認できる画面です。重要度や公開期間、固定表示、既読・未読などが分かりやすく表示されます。

### 主な機能
- お知らせ一覧の表示（タイトル・重要度・公開期間・固定表示・未読マーク）
- お知らせ内容の詳細表示
- 一括既読ボタン
- 表示更新ボタン
- ページネーション（1ページ10件）
- 未読のお知らせは赤丸で表示
- 固定フラグはピンアイコンで表示

## 画面項目
| #  | 項目名           | 必須 | 説明                                                                 |
|:--:|:-----------------|:----:|:-------------------------------------------------------------------|
| 1  | お知らせ一覧     |  -   | タイトル・重要度・公開期間・固定・未読マークを一覧表示します        |
| 2  | 重要度バッジ     |  -   | お知らせの重要度（高・中・低）をバッジで表示します                  |
| 3  | タイトル         |  -   | お知らせのタイトル（長い場合は省略＋ツールチップ）                  |
| 4  | 公開期間         |  -   | お知らせの公開開始日・終了日を表示します                            |
| 5  | 固定フラグ・未読マーク       |  -   | 固定されたお知らせはピンアイコンで表示され、未読のお知らせは赤丸で表示                              |
| 6  | 内容表示欄       |  -   | 選択したお知らせのタイトル・本文を詳細表示                          |
| 7  | ページネーション |  -   | 1ページあたりの表示件数やページ切替ができます                        |
| 8  | 一括既読ボタン   |  -   | 全てのお知らせを既読にします（確認ダイアログあり）                   |
| 9  | 表示更新ボタン   |  -   | お知らせ一覧を最新状態に更新します                                  |
| 10 | 閉じるボタン     |  -   | ダイアログを閉じます                                                |
