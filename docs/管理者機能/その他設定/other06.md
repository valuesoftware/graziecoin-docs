## 画面項目
|   #   | 項目名                             | 必須  | 説明                                                                                                                               |
| :---: | :--------------------------------- | :---: | :--------------------------------------------------------------------------------------------------------------------------------- |
|   1   | ユーザ更新フォーム有効スイッチ     |   -   | ユーザ更新フォームからのユーザ更新の[有効](#_5)・[無効](#_6)を切り替えるスイッチです                                                             |
|   2   | ユーザ更新URL                      |   -   | ユーザ更新フォームのURLを表示します                                                                                                |
|   3   | URLをコピーボタン                  |   -   | ボタンを押すとユーザ更新URLをクリップボードにコピーします                                                                          |
|   4   | URLを再発行するボタン              |   -   | ボタンを押すとユーザ更新URLをランダムに生成しなおします                                                                            |
