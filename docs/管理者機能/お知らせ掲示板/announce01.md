# お知らせ一覧画面

## ❓お知らせ掲示板とは

「お知らせ掲示板」は、管理者がシステム利用者に向けて、お知らせを公開できる機能です。


### お知らせ掲示板一覧画面の主な機能
- お知らせの新規追加
- お知らせ内容の編集
- お知らせの削除

## 画面項目
|   #   | 項目名                | 必須  | 説明                                                                 |
| :---: | :-------------------- | :---: | :------------------------------------------------------------------- |
|  1    | [お知らせ追加ボタン](announce02.md)    |  -    | 新しいお知らせを追加するダイアログを開きます                         |
|  2    | 件数表示              |  -    | 登録されているお知らせの総件数を表示します                           |
|  3    | お知らせ一覧テーブル  |  -    | お知らせのタイトル・本文・公開対象・期間・重要度などを一覧表示します |
|  4    | [編集ボタン](announce03.md)            |  -    | 選択したお知らせの内容を編集できます                                 |
|  5    | [削除ボタン](announce01.md#_9)            |  -    | 選択したお知らせを削除します（確認ダイアログあり）                   |
|  6    | ページネーション      |  -    | 1ページあたりの表示件数やページ切替ができます                        |
