## 導入後に行う作業
1. 組織を登録する
2. ユーザを登録する
    - ユーザ登録
    - CSVインポート
    - ユーザ登録フォーム
3. コインを補充する
    - コインの使い道
        - 表彰
        - 配布
        - スパグラの原資
