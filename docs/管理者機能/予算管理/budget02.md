## 画面項目
|   #   | 項目名       | 必須  | 説明                                       |
| :---: | :----------- | :---: | :----------------------------------------- |
|   1   | ユーザ名     |   -   | ユーザ名とID(メールアドレス)を表示します   |
|   2   | GRC残高      |   -   | ユーザが保有しているコイン残高を表示します |
|   3   | [閉じるボタン](budget01.md) |   -   | ボタンを押すと画面を閉じます               |
